COLOR PARSLEY REGEX



GUIDE:

  • GROUP 1: by itself takes 0-255 as INT and must be terminated with a comma or space. By itself generates greyscale by applying the value to all three r g b and alpha at 1. Leading rgb(a) and parenthesis are optional.
  • GROUPS 1 thru 4: G1-3 take 0-255 as INT, must be separated with a comma or space, G4 takes an INT percentage if terminated with % or a float 0 to 1. Leading rgb(a) and parenthesis are optional.
  • GROUPS 5 thru 8: takes three or four digit HEX string. Hash is optional. #afc or 1234. May have an optional space at the end only.
  • GROUP 9: By itself takes a two digit HEX string. Hash is optional. #ab By itself generates greyscale by applying the value to all three r g b and alpha at 1. I.e. #ab becomes ababab
  • GROUPS 9 thru 12: takes six or eight digit HEX string. Hash is optional. #abcdef or 12345678.
  • GROUPS 13 thru 17: "catch all" for various spaces. G13 takes a 3 to 5 character text string identifyer such as lch. The first character can NOT be [abcd]. Parenthesis are optional. Values can be anything from INT to percentage to float, there is minimal verification. Each value must be separated with a comma or space, except the last (alpha) which can use comma or a slash.



REGEX STRING

Here is the actual string for parsing various types of color strings with validation for "rgb()" and hex strings. Minimal checking for hsl/xyz/lchuv etc. type strings. For clarity it is broken into three pieces per the above flowchart. 


// First section, for rgb()
/(?:^(?:rgba?|)\(?\s?(?:(25[0-5]|2[0-4]\d|1?\d{1,2})(?:, ?| ))(?:(25[0-5]|2[0-4]\d|1?\d{1,2})(?:, ?| )(?:(25[0-5]|2[0-4]\d|1?\d{1,2}))(?:(?:,| \/| )\s?(?:(100%|\d?\d%|[01]?\.\d{1,22}|[01])))?(?: \)| |))?(?!\S)|

// Second section, for hex strings, either   abc or aabbcc
^(?:#|0x|)(?:(?:([\da-f])([\da-f])([\da-f])([\da-f])?)(?!\S)|(?:([\da-f]{2})(?:([\da-f]{2})([\da-f]{2})([\da-f]{2})?)?))(?!\S)|

// Third section, for hsl, hwb, lch, and anything as in stuvw(1,1,1,1)
^(?:([^r\d\W][a-z]{2,5})\(?\s?((?:\d{0,3}\.|)\d+%?)(?:, ?| )((?:\d{0,3}\.|)\d+%?)(?:, ?| )((?:\d{0,3}\.|)\d+%?)(?:(?:,| ?\/) ?((?:\d{0,3}\.|)\d+%?))?)(?: \)| |))$/i
